### Tech used
#### Frontend
-   React
-   Redux
-   Reactstrap/Bootstrap
-   FontAwesome
#### Backend
-   Express
-   PostgreSQL
-   ElephantSQL
-   Sequelize
-   Bcrypt, Passport, and JWT for authentication

### How to run
In the root directory run:
```sh
$ npm install
$ npm run client-install
$ npm run dev
```
