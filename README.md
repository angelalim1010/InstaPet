### Tech used
#### Frontend
-   React
-   Redux
-   Reactstrap/Bootstrap
-   Fontawesome
#### Backend
-   Express
-   Postgres
-   Sequelize
-   Elephantsql
-   Bcrypt, passport, and JWT for authentication

### How to run
In the root directory run:
```sh
$ npm install
$ npm run dev
```
